## 目的

* サーバー起動時のメインワールド → フラットなロビー
* ネザー・エンド → 作らない
* 別ワールド `/mv create test` → 通常地形のサバイバル用

## 1. server.properties の最初のワールドをロビー用に

`server.properties`

```properties
level-name=lobby
```

これで最初に読み込まれるワールド名が `lobby` になります。

---

既に world があるなら、必要に応じてバックアップしてから：

## 3. フラットワールド生成（ネザー・エンド無し）

サーバー起動後、OP権限で：

```mcfunction
/mv create lobby normal -g VoidGen
```

または単純フラットなら：

```mcfunction
/mv create lobby normal -t FLAT
```

* `VoidGen` → 完全な虚空ロビー（おすすめ）
* `FLAT` → 草ブロックの平坦世界

`server.properties`


Paperならさらに：

```yml

  allow-end: false
```

または Paper の worlds 設定で無効化。
