Minecraft の「ワンブロック（OneBlock）」や「フラット（スーパーフラット）」は、遊び方によって作り方が少し違います。
Java版前提で、今のPaperサーバー環境でも使える形でまとめるね。

---

# フラットワールドの作り方（超簡単）

## シングルプレイ

1. Minecraft起動
2. 「シングルプレイ」→「ワールド新規作成」
3. 「その他のワールド設定」
4. 「ワールドタイプ」を

   * `スーパーフラット`
     に変更
5. 作成

---

## サーバー（Paper）

`server.properties` を編集。

```properties
level-type=minecraft:flat
```

保存後、ワールドを作り直す。

### 注意

既存ワールドがあると反映されないので、

```txt
world
world_nether
world_the_end
```

を削除して再生成。

---

# ワンブロックの作り方

ワンブロックは公式モードではなく、主に

* 配布マップ
* プラグイン
* データパック

で作る。

---

# 一番簡単：配布マップを使う

## 手順

### 1. OneBlock配布マップをDL

有名なのは：

* IJAMinecraft版
* OneBlock Original

など。

---

### 2. zip解凍

中に

```txt
level.dat
region
datapacks
```

などがある。

---

### 3. サーバーへ入れる

Paperサーバー停止後、

```txt
/world
```

を削除して、

配布マップフォルダを `world` にリネームして配置。

---

### 4. サーバー起動

これで中央の1ブロックから始まる。

---

# 自作ワンブロック（超シンプル版）

コマンドだけでも可能。

## フラット世界を作る

```properties
level-type=minecraft:flat
```

---

## 初期地点にブロック設置

```mcfunction
/setblock 0 100 0 grass_block
```

---

## 下を奈落にしたい場合

スーパーフラット設定を空に近くする。

Java版なら：

```txt
minecraft:air;minecraft:the_void
```

系のカスタム生成を使う。

# Skriptで簡易OneBlock作る例

例えば中央ブロック破壊時：

```vb
on break of stone:
    if location of event-block is location(0, 100, 0):
        wait 1 tick
        set block at location(0,100,0) to random element out of stone, dirt, oak_log, diamond_ore
```

これだけでも「壊すと再生成」になる。
