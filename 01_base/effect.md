# エフェクト（Effect）[Java Edition]

### 移動速度上昇（Speed）

* 効果：移動速度が上昇する
* ID：speed
* コマンド：`/effect give @s minecraft:speed infinite 255 true`

### 移動速度低下（Slowness）

* 効果：移動速度が低下する
* ID：slowness
* コマンド：`/effect give @s minecraft:slowness infinite 255 true`

### 採掘速度上昇（Haste）

* 効果：採掘・攻撃速度が上昇する
