# Chunkyは何をする？

Chunkyは：

> 「チャンクを生成して保存する」

だけです。

つまり、

* 一度マップを作る
* ディスクに保存する
* 普段は読み込まない

という動き。

---

# `/forceload` との違い

## `/forceload`

```mcfunction id="tx7ydt"
/forceload add
```

これは：

> 常時チャンクを読み込み続ける

ので、

* Mob動く
* レッドストーン動く
* ホッパー動く

＝重い。

---

# Chunky

一方Chunkyは：

* 先に地形生成
* 保存だけ
* 普段は停止

なので軽い。

---

# 線を引くみたいにマップ作れる？

できます。

例えば：

## 四角形

```mcfunction id="t2mr3u"

chunky start
```

---

## 円形

```mcfunction id="z2lcl3"

chunky radius 3000
chunky start
```
