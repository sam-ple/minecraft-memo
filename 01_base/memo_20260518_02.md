# Skriptを書く

## oni.sk

```vbnet id="jlwmm3"
on damage:
    attacker is player
    victim is player

    if attacker's tool is netherite axe:

        if attacker is in team "oni":
            if victim is in team "runner":

                execute console command "team leave runner %victim%"
                execute console command "team join oni %victim%"

                broadcast "☠ %victim% became Oni!"
```

---

# これだけ

かなり簡単。
